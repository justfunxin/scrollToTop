# scrollToTop

### Useage:

```javascript
$('.scrollToTop').scrollToTop();
```
### Options:

```javascript
$.fn.scrollToTop.defaults = {
    container: window,
    target : 'html, body',
    speed : 800,
    distance: 0,
    top: 0
}
```
### Customized with metadata

```html
<a href="#" class="scrollToTop" data-scroll-target="#div1" data-scroll-speed="600" data-scroll-distance="100" data-scroll-top="10">TOP</a>
```
